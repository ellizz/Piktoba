# Piktoba

Piktoba is a simple Rock Paper Scissors game with funny references to my friends and other games.  
The game includes voice acting and light humor.

---

## Features

The game includes the following:

• Rock Paper Scissors gameplay  
• Funny references and inside jokes  
• Voice acting  
• Simple story presentation
